Go zipext
=========

[![Build Status](https://travis-ci.org/enr/go-zipext.png?branch=master)](https://travis-ci.org/enr/go-zipext)

Go library to manipulate zip files.

Import library:

```Go
    import (
        "github.com/enr/go-zipext/zipext"
    )
```
